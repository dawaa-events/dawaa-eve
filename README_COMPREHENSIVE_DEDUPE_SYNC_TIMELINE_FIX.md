# DAWAA Comprehensive Dedupe + Delete + Sync + Timeline Fix

## يشمل هذا الباتش

1. إزالة تكرار الضيوف:
   - التكرار الناتج من رقم الترتيب.
   - التكرار الناتج من الترتيب الأبجدي.
   - الدمج حسب: المناسبة + رقم الهاتف + الاسم بعد تنظيف المسافات والأحرف.

2. إصلاح الحذف:
   - حذف الضيف محليًا.
   - حذف كل نسخه المشابهة من Supabase.
   - منع رجوعه بعد Refresh أو Sync.

3. إصلاح المزامنة:
   - Supabase هو مصدر الحقيقة.
   - عرض الضيوف يتم بدون تكرار.
   - إضافة تنظيف تكرار من السيرفر.

4. النسخ الاحتياطي:
   - قبل أي حذف أو تنظيف، يتم حفظ Backup محلي في LocalStorage:
     `dawaa_backup_<timestamp>`

5. إصلاح Timeline:
   - لم يعد يعرض "لم يتم بعد" بشكل مضلل.
   - إنشاء الضيف إذا لا يوجد وقت: `وقت الإنشاء غير متوفر`
   - قراءة الرسالة إذا لا يوجد Read webhook: `لم يتم التعرف على وقت القراءة`
   - إرسال الدعوة إذا لم يحدث: `بانتظار الإرسال`
   - تسليم الرسالة إذا لم يحدث: `بانتظار التسليم`
   - تسجيل الرد إذا لم يحدث: `بانتظار الرد`
   - إضافة حالة رمادية/غير معروفة بدل حالة انتظار خاطئة للقراءة والإنشاء.
