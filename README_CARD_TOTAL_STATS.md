# DAWAA Card Total Stats Patch

هذا باتش واجهة فقط يعدل الإحصائيات العلوية لتُحسب حسب عدد البطاقات وليس عدد الأشخاص.

## مثال
إذا ضيف واحد لديه 6 بطاقات واختار تأكيد 4:
- إجمالي: 6
- حاضر: 4
- معتذر: 2
- لم يؤكد: 0

بدل:
- إجمالي: 1
- حاضر: 1
- معتذر: 0
