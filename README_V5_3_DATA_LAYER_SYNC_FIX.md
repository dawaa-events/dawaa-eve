# DAWAA V5.3 Data Layer Sync Fix

هذه النسخة مبنية على الملف الذي قلتي إنه مرفوع على Vercel ويعمل من الهاتف.

## المشكلة التي وجدتها
في بداية `app.js` كانت دوال المزامنة معطلة فعلياً:

- loadGuestsFromServer لا تسحب شيئاً
- saveGuestsToServer لا تحفظ شيئاً
- mergeGuestsFromServer لا تدمج شيئاً

لذلك الهاتف واللابتوب كانا يعتمدان على LocalStorage فقط، والمتصفح المتخفي يظهر فارغ.

## استبدلي هذه الملفات
- app.js
- styles.css
- api/guests-sync.js
- api/_lib/supabase.js
- api/_lib/meta.js
- api/webhook/meta.js

## ماذا تم إصلاحه؟
- تفعيل المزامنة الحقيقية عبر `/api/guests-sync`.
- تعطيل الكاش في طلبات المزامنة.
- أي إضافة أو استيراد ضيوف تحفظ في Supabase.
- أي جهاز يفتح الموقع يسحب الضيوف من Supabase.
- زر جانبي: تحديث البيانات.
- قبل الإرسال يتم سحب آخر بيانات من Supabase.
- رسالة فشل الإرسال أصبحت تعرض السبب الحقيقي بدل رسالة عامة.
- إصلاح فلترة الضيوف عند وجود رقم أو اسم ناقص.
