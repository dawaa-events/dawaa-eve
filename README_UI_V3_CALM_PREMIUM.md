# DAWAA UI v3 — Calm Premium

تم تطبيق التصميم المعتمد: هادئ، راقي، أبيض وبنفسجي ناعم، بدون عناصر ضخمة.

## الملفات
- app.js
- styles.css

## التغييرات
- حذف عبارة "بوابة الإدارة والعملاء".
- إصلاح فتح مساحة العمل كصفحة مستقلة `/admin/workspace`.
- تصميم Sidebar أبيض وهادئ مثل المرجع.
- كروت KPI هادئة وصغيرة.
- Timeline أفقي واضح.
- تحسين الضيوف والجداول والحالات.
- ألوان الحالات:
  - أخضر للحاضر/تم التأكيد
  - أصفر لغير مؤكد
  - أحمر للمعتذر
  - أزرق للإرسال/المعلومات
- تصميم Login أقرب للمرجع وبهدوء.

## لم يتم تعديل
- api/
- webhook
- send-invitations
- guests-sync
- supabase
- vercel
- Environment Variables

## ملاحظة
تم تعديل app.js فقط في الواجهة والتنقل، وليس في منطق الإرسال أو المزامنة.
