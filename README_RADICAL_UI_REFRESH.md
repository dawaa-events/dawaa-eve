# DAWAA Radical UI Refresh Patch

باتش تصميم جذري للواجهة فقط.

## الملفات
- app.js
- styles.css

## ماذا يغير؟
- يحول شكل النظام إلى Workspace أفخم.
- يحسن كروت الضيوف.
- يحسن كروت المناسبات.
- يحسن التبويبات لتصبح Action Dock.
- يحسن الجوال.
- يضيف زر دخول سريع إلى صفحة الإرسال من كرت المناسبة إذا كان النمط موجوداً في الملف.

## لم يتم تعديل
- api/
- webhook
- send-invitations
- guests-sync
- supabase
- vercel
- Environment Variables

## ملاحظة
هذا تحديث تصميم فقط ولا يغير منطق الإرسال أو المزامنة.
