# DAWAA UI v2 Layout Rebuild

هذا باتش واجهة حقيقي وليس صورة.

## الملفات التي تتغير
- app.js
- styles.css

## ماذا يغير؟
- صفحة تسجيل دخول جديدة بالكامل:
  - مرحباً بك
  - قم بتسجيل دخولك
  - إذا كنت لا تمتلك حساباً يرجى التواصل مع إدارة دعوة
  - تصميم Split Layout راقي وخفيف

- إلغاء فكرة فتح مساحة المناسبة في قائمة جانبية:
  - الضغط على "فتح مساحة العمل" يفتح صفحة مستقلة `/admin/workspace`

- إضافة مساحة عمل للمناسبة:
  - Countdown
  - Timeline أفقي للمهام
  - أزرار واضحة: الضيوف، الإرسال، الرسائل، التقارير
  - KPIs حسب البطاقات

- إضافة تصنيف القائمة في الضيوف/الإرسال:
  - قائمة العروس
  - قائمة المعرس
  - قائمة معدلة
  - قائمة احتياط

- تصغير الخطوط والأيقونات وتصميم أهدأ.

## لم يتم تعديل
- api/
- webhook
- send-invitations
- guests-sync
- supabase
- vercel
- Environment Variables

## ملاحظة
تم تعديل app.js فقط في واجهة العرض والتنقل، وليس في منطق الإرسال أو المزامنة.
