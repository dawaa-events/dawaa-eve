# DAWAA Hard Clean Import + Duplicates Fix

## المشكلة
كانت تظهر ضيوف بأسماء استفهامات/رموز وتكرارات كثيرة بسبب سجلات تالفة محفوظة في Supabase من استيراد Excel سابق.

## الإصلاح الأقوى
- منع عرض أي ضيف تالف في صفحة الضيوف.
- منع استيراد أي صف اسمه تالف أو رقمه ناقص.
- منع تكرار نفس الاسم + نفس الرقم في نفس المناسبة.
- زر تنظيف التالف/المكرر يحذف التالف والتكرارات من Supabase.
- المزامنة تتجاهل السجلات التالفة حتى لو بقيت قديمة.
- لا يحذف اختلاف الأسماء على نفس الرقم.

## بعد الرفع
1. افتحي صفحة الضيوف.
2. اضغطي زر: تنظيف التالف/المكرر.
3. بعد التنظيف ارفعي ملف Excel الصحيح.
4. لا ترفعين نفس الملف أكثر من مرة؛ ولو رفعتيه بالغلط، سيتجاهل المكرر.
