# DAWAA Event Countdown Patch

باتش واجهة فقط يضيف عداد تنازلي للمناسبة وخانة الخطوة القادمة.

## الملفات
- app.js
- styles.css

## ماذا يظهر؟
- تبقى كذا يوم على المناسبة.
- الخطوة القادمة:
  - إرسال الدعوات
  - متابعة الردود
  - إصدار البطاقات
  - إرسال البطاقات
  - تنبيه عاجل
