# DAWAA V4 Complete Fix Pack

نسخة تجمع الأشياء التي كانت جزئية أو واجهة فقط وتحوّلها لتطبيق فعلي قدر الإمكان بدون لمس ملفات الإرسال.

## الملفات التي تستبدل
- app.js
- styles.css

## ما تم تثبيته
- فتح مساحة العمل كصفحة مستقلة وليس Drawer.
- فتح تفاصيل الضيف كصفحة مستقلة وليس Drawer.
- Countdown مباشر بالأيام والساعات والدقائق والثواني.
- آخر تحديث يظهر بالوقت.
- Timeline أفقي للمناسبة.
- Timeline أفقي لتفاصيل إرسال الضيف.
- تصنيف القوائم فعلياً عند إضافة/استيراد الضيوف:
  - قائمة العروس
  - قائمة المعرس
  - قائمة معدلة
  - قائمة احتياط
- إضافة تصنيف القائمة داخل نافذة إضافة الضيف.
- صفحة صلاحيات العميل داخل مساحة المناسبة:
  - السماح بالإضافة
  - السماح بالتعديل
  - السماح بالحذف
  - السماح بقائمة الاحتياط
  - حد البطاقات المسموح للعميل
- بوابة العميل تعرض حد البطاقات والمتبقي.
- إذا لا توجد صلاحية، تظهر رسالة طلب إذن من إدارة دعوة.
- تحسين ألوان الحالات.
- تحسين تجربة الجوال والتصميم الهادئ.

## لم يتم لمس
- api/
- webhook
- send-invitations
- guests-sync
- supabase
- vercel
- Environment Variables

## ملاحظات مهمة
- Agent تحليل الرسائل لم يتم إدخاله هنا لأنه يحتاج Backend/Workflow مستقل.
- تفعيل صورة الدعوة يعتمد على قالب Meta وملفات API، لذلك لم يتم تعديله في هذا الباتش حتى لا ينكسر الإرسال.
